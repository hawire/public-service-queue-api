# public-service-queue-api
Public Service Queue Management API built with Django REST Framework
## Features (Planned)
- User authentication (Admin & Staff)
- Citizen registration
- Service management
- Queue ticket generation
- Ticket status updates (waiting, serving, completed)
- Queue analytics
- Role-based access control

## Tech Stack
- Python
- Django
- Django Rest Framework
- PostgreSQL / SQLite
- JWT Authentication

## ERD
Includes Users, Citizens, Services, and Queue Tickets.
